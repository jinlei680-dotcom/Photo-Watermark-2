# 需求文档：图片拍摄日期水印批量添加工具（图形化界面）

一、目标与范围
- 目标：提供一个图形化界面程序，批量读取图片的 EXIF 拍摄时间（取年月日），将该日期作为文本水印叠加到图片上，并输出到原目录的子目录“原目录名_watermark”中。
- 范围：面向个人或工作场景下的照片批处理，支持基本水印样式配置（字体大小、颜色、位置）。

二、术语与约定
- EXIF 拍摄时间字段：优先使用 DateTimeOriginal；如不存在，可使用 CreateDate 或其他等效拍摄时间字段；如果所有拍摄时间字段缺失，则该图片视为“不含拍摄时间”，按定义的错误处理策略处理。
- 年月日格式：默认格式为 YYYY-MM-DD（例如：2024-06-18）。
- 位置枚举：左上角、居中、右下角。

三、使用场景与用户流程
1) 用户启动程序，进入主界面。
2) 用户输入一个路径：
   - 支持选择“目录路径”：处理该目录下所有图片文件（不递归子目录，除非后续扩展）。
   - 支持选择“单个图片路径”：仅处理该图片；输出目录依然为其所在目录的“目录名_watermark”子目录。
3) 用户设置水印参数：
   - 字体大小：整数，单位可视为像素。
   - 字体颜色：支持十六进制颜色（如 #000000、#FFFFFF）或提供常用颜色选择。
   - 水印位置：左上角、居中、右下角三选一。
4) 用户点击“开始处理”，程序批量生成水印图片：
   - 对每个图片读取 EXIF 拍摄时间（取年月日），生成文本水印。
   - 将带水印的新图片保存到“原目录名_watermark”子目录，文件名与原文件一致。
5) 处理完成后，界面提示成功并显示统计信息（成功处理数、跳过数、错误数）。

四、功能性需求
- 路径输入
  - 提供文件/目录选择器。
  - 自动识别用户选择的是单个文件还是目录。
- 图片扫描
  - 当输入为目录时，枚举该目录下的所有图片文件（建议支持扩展名：.jpg/.jpeg/.png；后续可扩展其他格式）。
  - 当输入为单文件时，仅处理该文件。
- EXIF 读取与日期提取
  - 优先读取 DateTimeOriginal；如缺失，再尝试 CreateDate 等。
  - 日期提取方式：从拍摄时间中解析出年月日，格式化为 YYYY-MM-DD。
  - 若无法读取拍摄时间：该图片标记为“跳过”，记录在处理结果中。
- 水印绘制
  - 文本内容：解析出的日期字符串 YYYY-MM-DD。
  - 字体大小：用户设置值。
  - 字体颜色：用户设置值。
  - 位置：按用户选择在图片上绘制（左上角、居中、右下角）。
  - 不修改原图，输出为新图。
- 输出与目录结构
  - 输出目录：原目录下新建子目录“原目录名_watermark”。
  - 文件命名：沿用原始文件名。
  - 目录已存在时：允许复用目录并覆盖同名输出文件（简化流程）。
- 结果与提示
  - 显示进度与结果统计（成功数、跳过数、错误数）。
  - 对错误（无法读取 EXIF、非图片文件、写入失败）进行提示。

五、非功能性需求
- 兼容性：至少在 macOS 环境正常运行（当前开发环境）。
- 性能：对常见目录（数百张照片）可在合理时间内完成；性能不作过度优化要求。
- 稳定性：遇到异常文件不影响整体任务；跳过并继续处理其他图片。
- 安全性：不修改原文件，不上传任何文件到网络。

六、用户界面要求
- 清晰的输入区域与文件/目录选择按钮。
- 水印参数设置区域：
  - 字体大小（输入框或滑条）
  - 字体颜色（颜色选择器或输入框）
  - 位置选择（单选框：左上角、居中、右下角）
- 操作按钮：开始处理、取消（可选）。
- 输出信息展示：处理进度与完成统计。
- 错误提示：以列表或弹窗提示问题文件及原因。

七、文件与格式支持
- 默认支持 JPG/JPEG；可兼容 PNG（若无 EXIF 则跳过）。
- 后续可扩展 HEIC、TIFF 等格式（非本次必需）。

八、异常与错误处理
- 无效路径：提示用户并不开始处理。
- 非图片文件：跳过并记入统计。
- 无 EXIF 拍摄时间：跳过并记入统计。
- 输出目录创建失败/写入失败：提示并记入统计。
- 覆盖策略：目标同名文件允许覆盖。

九、验收标准
- 当用户选择某目录：
  - 程序能识别并处理该目录下的所有符合支持格式的图片。
  - 正确读取 EXIF 拍摄时间并按 YYYY-MM-DD 绘制到指定位置。
  - 在原目录下成功创建“原目录名_watermark”子目录，输出文件命名与原文件一致。
  - 原始图片未被修改。
  - 能显示成功、跳过、错误的统计信息。
- 当用户选择单张图片：
  - 程序能对该图片正确绘制水印并输出到其所在目录下的“目录名_watermark”子目录。

十、后续可扩展（非本次必需）
- 预览功能：在处理前预览水印效果。
- 自定义日期格式：YYYY/MM/DD、YYYY年MM月DD日等。
- 透明度设置与字体选择。
- 递归处理子目录。
- 批量位置规则（如相对边距）。
- 多语言界面。
